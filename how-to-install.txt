Method 1: 
    1.	First Login to admin panel 
    2.	Go to : (Preference) > (Configuration) 
        Set, Debug Mode = 1
    3.	Click on (Application) tab
    4.	Click on (Install New Component)
    5.	Select blog.zip file ( Click on the Floppy Disk image on right side of the upload file input box)
    6.	Click on Install Button. 
    7.	After upload, this Blog component will appear in the list.
        Now click on "Activate" link blow description. 
        
Method 2: 
    1.	First Login to admin panel 
    2.	Go to : (Preference) > (Configuration) 
        Set, Debug Mode = 1
    3.	Unzip blog.zip.
    4.	Copy "blog" folder in “component” directory in your project
    5.	In Admin panel go to application tab and find blog component
    6.	Click on Activate Link. 
	
Important: After Activation of Blog component a new table supposed to be created named "blogcomments". If it's not found
    - First check debug mode should be 1 (Step 2) during activation
    - Check database user access
